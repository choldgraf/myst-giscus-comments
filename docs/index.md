# Welcome

This page demonstrates the **myst-giscus** plugin, which adds [Giscus](https://giscus.app) comments to every page of a MyST site.

Giscus uses GitHub Discussions as a comments backend — each page gets its own discussion thread, mapped by URL path.

## How it works

The plugin uses MyST's anywidget support to inject a giscus iframe at the bottom of every page. No theme modifications needed.
